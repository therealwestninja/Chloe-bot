# Chloe-bot
A free, locally hosted Perchance-based Discord bot (Greasemonkey userscript bridge) which runs in your browser.

## How-to

1) Required:
- Google Chrome
- Greasemonkey / Tampermonkey Chrome extension installed
- Userscript running.
- Have the Perchance generator page open.

2) Set up a new Discord Bot and invite it to your server.

3) Set up the Bot page:

- Run Discord in Developer mode (Settings cog, Developer > "Developer Mode" enabled.)
In Discord, right click the #Channel you want the bot to monitor in your channel list, click "Copy Channel ID", and paste it into the Generator's setup and click "Set Channel".

TIP:
 If you're adding the bot to an existing channel or running it for the first time, click "Backfill history" to populate the bot with context memory

4) Set any additional options, then click Start to run the bot.
